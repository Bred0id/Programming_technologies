# Задание по git

> Технологии программирования, 2024г

## Часть 3.

Завершаем работу с мерж-конфликтами. В предыдущих двух частях мы учились разрешать мерж-конфликты, полученные в различных условиях, однако не все варианты были покрыты. В этой части вы должны будете создать мерж-конфликт, отличный ото всех, встреченных ранее.

Используя дополнительную ветку `6`, сделайте так, чтобы при после вызова на текущей ветке (`5`)

```shell
git merge 6
```

произошел мерж-конфликт.
