# Задание по git

> Технологии программирования, 2024г

## Часть 1.

Слейте ветку `2` в текущую ветку (`1`)

При сливании возникнет конфликт. Разрешите его: в одном файле оставьте изменения из обоих веток, в итоговой версии файла числа должны идти по убыванию; другой же, который был удален на одной из веток, удалите.

Мерж-коммит должен иметь название `3`.
